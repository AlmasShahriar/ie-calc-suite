⏱ Time Study Calculator
Enter observed times per element. Get Normal Time & Standard Time with PFD allowance.
Normal Time = Observed Time × (Rating Factor / 100)
Standard Time = Normal Time × (1 + Allowance / 100)

Work Elements

OUTPUT
Element Obs. Times Mean (s) Normal Time (s) Std Time (s)
⚖ Line Balancing Calculator
Enter task times and number of workstations. Get efficiency, idle time, and bottleneck identification.
Cycle Time = Production Time / Required Output  |  Min Stations = Σ Task Times / Cycle Time
Efficiency = (Σ Task Times) / (N × Cycle Time) × 100  |  Balance Delay = 100 − Efficiency

Tasks / Operations

OUTPUT
Task Time (min) % of Cycle Status
📦 ABC Analysis Classifier
Paste item data (one per line: Item, Annual Usage, Unit Cost). Auto-classifies by annual value.
Annual Value = Annual Usage × Unit Cost
Class A = Top items contributing ~70% of total value  |  B = Next ~20%  |  C = Remaining ~10%
C = remainder

// Format: Name, Quantity, Unit Cost — one item per line

OUTPUT
